Локализация главного меню окна (в перспективе и контестного меню окна). Локаль берется из атрибута тега html lang например если в dfitv теге html <html lang="ru"> локаль будет ru В папке приложения ищется файл /js/ru.json Пока поддерживаются только "плоские" файлы Пример файла локализации главного меню окна 
{
	"File":"Файл", 
	"Open Ctrl+O": "Открыть Ctrl+O",
	"Save Ctrl+S": "Сохранить Ctrl+S",
	"Save as... Ctrl+Shift+S": "Сохранить как... Ctrl+Shift+S"
}
ТОгда, если ваш штвучюреьд содержит html 

	
		Open Ctrl+O
		Save Ctrl+S
		Save as... Ctrl+Shift+S
Пункты меню будут переименованы. Скоро: Будут автоматически переводиться также аттрибуты value тегов input[type="button"] и содержимое см. tools/j.js
